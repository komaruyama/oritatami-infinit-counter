These certificates have been generated by a software, which is written in Golang and you can download from https://github.com/komaruyama/certificatesOfInfiniteCounter/.

How to run this program

  1. Download the program from the URL above.
  2. Install a go compiler from https://golang.org/doc/install.
  3. Compile the program with the following command:

    $ go build main.go

  4. Run the compiled program.

    Unix OS: $ ./main

    Windows OS: Running by the created .exe file.

  5. The program outputs certificate svg files into "output" folder.
